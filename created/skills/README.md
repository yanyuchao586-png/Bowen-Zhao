# Created Skills

这里存放自己创建、改造或开发的 Skill。

## Index

| Skill | Description | Entry |
|---|---|---|
| `super-background` | 10 维客户背调、决策人图谱、聊天输出与 Word 报告生成 | `super-background/SKILL.md` |

每个 skill 目录建议包含：

- `SKILL.md`: 主要协议和执行流程。
- `README.md`: 面向 GitHub 访问者的说明页。
- `manifest.json`: 元数据、触发词和依赖。
- `examples/`: 示例输入或报告 payload。
- `references/`: 可选参考资料。
