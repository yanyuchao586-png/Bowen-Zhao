## Skill Folder Format / Skill 目录建议

- `SKILL.md`: 主要协议和执行流程。Main protocol and execution workflow.
- `README.md`: 面向 GitHub 访问者的说明页。Public GitHub-facing introduction.
- `manifest.json`: 元数据、触发词和依赖。Metadata, triggers, and dependencies.
- `examples/`: 示例输入或报告 payload。Example inputs or report payloads.
- `references/`: 可选参考资料。Optional references and implementation notes.
