# Created / 自建内容

这里存放自己创建、改造或开发的 Skill 与提示词。
This folder contains original, customized, or actively developed skills and prompts.

## Sections / 分类

- `skills/`: 自己创建或改造的 Skill。Original or customized skills.
- `prompts/`: 自己创建或整理的提示词。Original or refined prompts.

## Recommended Format / 推荐格式

每条内容建议包含用途、使用方法、示例和版本记录。
Each item should include purpose, usage instructions, examples, and version history.

适合开源传播的内容还应补充英文标题、英文关键词、许可证和截图。
For better GitHub discovery, add English titles, English keywords, license notes, and screenshots.
