## How To Use / 如何使用

1. 打开你感兴趣的目录。Open a prompt or skill folder.
2. 阅读该目录下的 `README.md`。Read its `README.md` first.
3. 复制 `prompt.md` 或 `SKILL.md`。Copy `prompt.md` or `SKILL.md`.
4. 粘贴到 Codex、ChatGPT、Claude、Cursor、v0、Manus 等 AI 工具中。Paste it into Codex, ChatGPT, Claude, Cursor, v0, Manus, or another AI coding tool.
5. 按你的项目技术栈、品牌文案和素材链接做二次修改。Customize the stack, brand copy, assets, and project constraints.
