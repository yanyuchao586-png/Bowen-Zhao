# Bowen Zhao

这个仓库用于公开收藏和整理我看到的、自己创建或开发的 Skill 与提示词。

## 目录结构

- `collected/`: 收藏的 Skill 或提示词。
- `created/`: 自己创建、改造或开发的 Skill 与提示词。

每个目录下都可以继续按 `skills/`、`prompts/` 或具体主题拆分。新增内容时，建议同时记录来源、用途、适用场景、许可证或授权信息。

## 收录建议

- 收藏内容请尽量保留原作者、原链接和许可证信息。
- 自建内容请写清楚使用方法、输入输出示例和维护状态。
- 不要提交 API Key、Token、账号密码或其他敏感信息。
