## Featured

### super-background

`super-background` 是一个 10 维客户背调 skill，用于从公司名、官网 URL 或线索截图出发，输出经过来源验证的客户全息档案、决策人图谱，并可按内置 Word 模板生成报告。

- 位置：`created/skills/super-background/`

- 公开页：`created/skills/super-background/README.md`
- 关键词：`super background`、`超级背调`、`客户背调`、`Customer Due Diligence`、`B2B Recon`

## Repository Structure

- `collected/`: 收藏的 Skill 或提示词。
- `created/`: 自己创建、改造或开发的 Skill 与提示词。

## Publishing Rules

- 收藏内容保留原作者、原链接和许可证信息。
- 自建内容写清楚用途、触发方式、安装方式、示例和维护状态。
- 不提交 API Key、Token、账号密码或其他敏感信息。
- 开源内容默认使用 MIT License；第三方素材或品牌资产如有单独授权，以其原授权为准。
