## How To Use

1. 打开某个 prompt 目录。
2. 先看 `README.md` 了解它适合什么项目。
3. 打开 `prompt.md`，复制完整提示词。
4. 粘贴到 Codex、ChatGPT、Claude、v0、Cursor 或 Manus。
5. 如果你是在已有项目中使用，先补一句：`请遵循当前项目的技术栈、目录结构和已有设计规范。`
6. 运行生成结果，检查依赖、移动端布局、动效性能和外部素材链接。

## Notes

- 预览素材来自对应来源网站，仅用于帮助理解 prompt 效果。
- 使用第三方代码、组件、字体、图片、视频或动效时，请遵守原网站许可证和授权条款。
- 不要把 API Key、Token 或私有业务数据粘进 prompt。
