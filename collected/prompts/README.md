## How To Use / 如何使用

1. 打开某个 prompt 目录。Open a prompt folder.
2. 先看 `README.md` 了解它适合什么项目。Read `README.md` to understand the use case.
3. 打开 `prompt.md`，复制完整提示词。Open `prompt.md` and copy the full prompt.
4. 粘贴到 Codex、ChatGPT、Claude、v0、Cursor 或 Manus。Paste it into Codex, ChatGPT, Claude, v0, Cursor, or Manus.
5. 如果你是在已有项目中使用，先补一句：`请遵循当前项目的技术栈、目录结构和已有设计规范。`
   If using an existing project, add: `Please follow the current project's tech stack, folder structure, and design system.`
6. 运行生成结果，检查依赖、移动端布局、动效性能和外部素材链接。Run the result and verify dependencies, mobile layout, animation performance, and external asset links.
